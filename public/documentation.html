MyFlix

For this project, I am going to build the server-side component of a “movies” web application. This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Return a list of all movies to the users /movies GET None None A JSON object with data about all movies
Return data about a single movie by title to the user /movies/[title] GET :title None A JSON object with data about the selected movie containing description, genre, director, and image URL. Example: 
            
{
  Title: '',
  Description: '',
  Year: '',
  Genre: '',
  Director: {
    name: '',
    birth: '',
    death: '',
  }
  Actors: ''
}
Return data about a genre (description) by name/title /movies/genres/[name] GET [name] None A JSON object with data about the selected genre. Example: 
            
{
  genre: 'Action'
}
Return data about a director (bio, date of birth) by name /movies/directors/[name] GET :name none A JSON object with data about the director. Example: 
            
{
  name: 'Denis Villeneuve',
  birth: 'October 03, 1967',
  death: '-'
}
Allow new users to register /users POST None A JSON object with data about the user to add. Example: 
            
{
  username: 'peterson',
  password: 'mr.peter',
  email: 'peterson@gmail.com'
  favorites: [
    'The Dark Knight',
    'Thor: Ragnarok',
    'Interstellar'
  ]
};
A JSON object with data about new user including ID. Example: 
            
{
  id: '1',
  username: 'peterson',
  password: 'mr.peter',
  email: 'peterson@gmail.com'
  favorites: [
    'The Dark Knight',
    'Thor: Ragnarok',
    'Interstellar'
  ]
};
Allow users to update their user info (username) /users/:username PUT :username A JSON object with new username. Example: 
            
{
  username: 'jacob'
}
A JSON object with the updated user profile. Example: 
            
{
  id: '1',
  username: 'jacob',
  password: 'mr.peter',
  email: 'peterson@gmail.com'
  favorites: [
    'The Dark Knight',
    'Thor: Ragnarok',
    'Interstellar'
  ]
}
Allow users to add a movie to their list of favorites /users/:username/:movie POST :username, :movie None A message saying that the movie was added
Allow users to remove a movie from their list of favorites /users/:username/:movie DELETE :username, :movie None A message saying that a movie has been deleted
Allow existing users to deregister /users/:username DELETE :username None A message saying the user has been deregistered
